# 使い方

## 概要
- 五十音表を編集して、辞書内で変換可能な単語ペアを見つけます。
- 編集は「五十音グリッド」または「CSV 直接編集」で行えます（両方は自動で同期されます）。

## アクセス
- https://github.Kaki256.io/word_shifter

## 画面の見方と操作

- 辞書を選択: 画面上部のドロップダウンから対象の辞書を選びます。選択後に「読み込み」ボタンで反映します。
- 入力方法: 「五十音グリッド」または「CSV 直接編集」を選べます。
	- 五十音グリッド: 画面に並んだマスに直接文字を入力します。
	- CSV 直接編集: テキスト形式の CSV を直接編集できます。編集すると自動でグリッドに反映されます。
- 無効マス（`＃`）: グリッド内の `＃` は編集不可です。これはキー側で固定されています。

## 変換の実行
- 辞書を選んだら「変換可能な単語ペアを探す」ボタンを押します。
- 検索結果は下部に一覧表示されます。表示はページ分割されます（1ページあたり 10 組）。
- ページ移動: 「-100 / -10 / 前 / 次 / +10 / +100」ボタンで素早く移動できます。

## ダウンロード
- `alt.csv をダウンロード` ボタンで現在のマップを CSV ファイルとして取得できます。
- `結果をCSVでダウンロード` ボタンで検索結果（全件）をダウンロードできます。

## その他
- 編集内容はブラウザの `localStorage` に自動保存されます（同じブラウザで再度開くと保存内容が復元されます）。
- 何か動作がおかしい場合は、ブラウザをリロードしてから再試行してください。
